实现函数 int sqrt(int x).

计算并返回x的平方根（向下取整）



示例1：

## 输入

```
2
```

## 输出

```
1
```



方法一：库函数

````
public class Solution {
    public int sqrt(int x) {
        return (int) Math.sqrt(x);
    }
}
````



方法二：

````
public class Solution {
    public int sqrt(int x) {
        int i = 1;
        int res = 0;
        while (x >= 0) {
            x -= i;
            res++;
            i += 2;
        }
        return res - 1;
    }
}
````
