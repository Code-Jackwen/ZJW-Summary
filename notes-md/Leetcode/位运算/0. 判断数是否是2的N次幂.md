使用位运算判断一个数是否是  2的n次幂。



方法一：

````java
public static boolean isPower(int n){
    if (n < 1) return false;
    return (n&(n-1))==0;
}
````

方法二：

````java
public static boolean isPower(int n){
    int i=1;
    if (n < 1) return false;
    while (i <= n){
        if (i == n){
            return true;
        }
        i <<= 1;//1  10  100 1000 1000
    }
    return false;
}
````
